# Examples

Ready-to-use shell scripts demonstrating common Social Post Flow CLI workflows.

| File | What it does |
|------|--------------|
| [`post-everywhere.sh`](post-everywhere.sh) | Post the same content to every text-capable profile the user has connected. |
| [`schedule-week.sh`](schedule-week.sh) | Add a week's worth of posts to the user's queue. |
| [`github-actions.yml`](github-actions.yml) | Auto-post to socials when a blog post is merged to your `main` branch. |

All examples assume `spf` is installed and `SOCIALPOSTFLOW_API_TOKEN` is set, or that `spf setup --key …` has already been run.
